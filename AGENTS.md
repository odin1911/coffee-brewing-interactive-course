# AGENTS.md

## 工作语言与方式

- 主对话使用简体中文。
- 不确定时先调查，再下结论。
- 改动前先找仓库里的相似实现，优先沿用现有模式。
- 小步前进；每次改动后运行与改动范围匹配的最小验证。
- 选择稳定、可维护的方案，不引入无必要复杂度。

## 必读文档

- 项目说明与文件职责：`README.md`。
- 项目硬约束：`docs/skills/Contract.md`。

## 禁止事项

- 不要绕开已有 skill 流程重新发明。
- 不要让实现先于文档长期漂移；任何使必读文档失真的改动，必须在同一批次按 `docs/skills/Contract.md` 的“活文档同步”要求更新。
